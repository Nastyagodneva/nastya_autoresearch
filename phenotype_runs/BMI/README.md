# Phenotype workspace: **BMI**

Created from `phenotype_runs/base_example/`. Phenotype-specific settings are in **`config.json`** (`target`, `session_name`, etc.).

Run optimization from this directory:

```bash
UV_PROJECT_ENVIRONMENT=/path/to/.venv uv run agent_loop.py
```

See `program.md` for agent rules. To refresh the template, copy from `../base_example/` again.
